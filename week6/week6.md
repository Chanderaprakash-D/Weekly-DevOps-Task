Week 6 – Terraform & Cloud Basics
Terraform
Terraform is an Infrastructure as Code (IaC) tool used to automate infrastructure.
`terraform init` – Initialize
`terraform plan` – Preview
`terraform apply` – Create resources
State & Backend
State file: Tracks created resources (`terraform.tfstate`)
Backend: Stores state file (local / remote like S3)

Cloud Basics
Cloud provides servers, storage, and services over the internet.
IaaS – EC2
PaaS – Platform for apps
SaaS – Ready software

AWS Concepts
EC2 – Virtual server
AMI – Template for EC2
Region & AZ – Location & data centers
